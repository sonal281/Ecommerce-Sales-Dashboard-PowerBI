# 📊 E-Commerce Sales Dashboard | Power BI

## 📌 Project Overview

This project presents an interactive E-Commerce Sales Dashboard built using Power BI. The dashboard analyzes sales performance, profit trends, customer behavior, payment methods, and regional sales insights.

The project demonstrates practical business intelligence and data visualization skills used in Data Analyst roles.

---

## 🛠️ Tools Used

* Power BI
* DAX
* Data Visualization
* CSV Dataset
* GitHub

---

## 📊 Dashboard Features

* Total Sales KPI
* Total Profit KPI
* Total Quantity KPI
* Average Order Value (AOV)
* Sales by State
* Sales by Customer
* Category Analysis
* Payment Mode Analysis
* Monthly Profit Trends
* Sub-Category Profit Analysis
* Interactive Filters & Slicers

---

## 📈 Key Insights

* Maharashtra generated the highest sales amount.
* Clothing category contributed maximum quantity sold.
* COD was the most preferred payment method.
* Monthly profit trends showed seasonal fluctuations.
* Printers generated the highest sub-category profit.
